# Website Travel Umroh Haji

## Preview Website

![Preview Website](preview4.png)

## Deskripsi

Website landing page untuk jasa travel umroh dan haji yang menampilkan informasi paket perjalanan, galeri perjalanan, dan kontak layanan.

Website ini dibuat untuk membantu jasa travel memiliki media promosi online yang sederhana.

## Fitur Website

- Landing page travel
- Informasi paket umroh
- Galeri perjalanan
- Testimoni
- Google Maps lokasi kantor
- Tombol WhatsApp konsultasi

## Teknologi yang digunakan

- HTML
- Tailwind CSS
- JavaScript
- GitHub Pages
